🔬 微觀視角：粒子與熱脹冷縮
低溫 ❄️
粒子間距縮小
→ 體積變小
室溫 🌤️
粒子間距適中
→ 正常體積
高溫 🔥
粒子間距增大
→ 體積變大
🔬 科學原理（有深度的補充）
物質由微小粒子組成，粒子不停振動：
• 溫度高 → 粒子振動幅度加大 → 間距增加 → 體積膨脹
• 溫度低 → 粒子振動幅度減小 → 間距縮小 → 體積收縮
• 特例：水在 0~4°C 是「熱縮冷脹」→ 湖泊冬天表面先結冰，底部保持液態保護水中生物！

🔥 三大傳熱方式比較
🍳
傳導 Conduction
主要發生在：固體
傳播方向

需要介質？
需要物質

📋 單元重點總整理
🌡️
熱脹冷縮
  • 氣體脹最多 > 液體 > 固體脹最少（但力量最強！）
  • 受熱 → 粒子間距增大 → 體積膨脹
  • 遇冷 → 粒子間距縮小 → 體積收縮
  • 例外：水在0~4℃是「熱縮冷脹」
🔥
傳導（固體）
  • 熱由高溫傳到低溫（固體為主）
  • 金屬傳熱快（銀>銅>鋁>鐵）
  • 非金屬傳熱慢（塑膠、木材、布、空氣）
  • 應用：鍋具設計（金屬鍋身+非金屬握把）
🌊
對流（液體/氣體）
  • 溫度高 → 液/氣密度小 → 上升
  • 溫度低 → 液/氣密度大 → 下降
  • 應用：冷氣裝上方、電暖器放地板、氣炸鍋
  • 沒有流動 = 沒有對流！
☀️
輻射（不需介質）
  • 不需要任何物質，真空中也能傳熱
  • 以電磁波（紅外線）傳播，光速！
  • 深色吸熱多；淺色（白色）反射多
  • 應用：遮陽傘、白色太空衣、深色集熱板
🧊
保溫與散熱
  • 保溫 = 阻隔或減緩熱的傳播
  • 散熱 = 加快熱的傳播速度
  • 保溫瓶：蓋（傳導對流）、內膽拋光（輻射）、真空（傳導對流）
  • 都市熱島效應：深色建材+缺少綠地導致城市過熱
